Circular Dependencies

No circular compile-time dependencies detected. All module dependencies form a directed acyclic graph (DAG). Runtime dependencies exist between managers (e.g., SshSessionManager and ConnectionPool) via Arc/Mutex, but these are intentional runtime patterns, not compile-time cycles.

Viewing the DOT File

To generate a visual graph from the dependency-graph.dot file, install Graphviz and run:
dot -Tsvg dependency-graph.dot -o dependency-graph.svg

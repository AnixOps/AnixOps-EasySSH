### 【添加第一台服务器】(1:30 - 4:00)

**[章节标题卡]**
"添加第一台服务器"

**[画面]**
1. **启动添加流程** (1:30 - 1:45)
   - 点击 [+ 添加服务器]
   - 弹出添加对话框
   - 高亮三个步骤指示器

2. **步骤一：基本信息** (1:45 - 2:30)
   - 名称字段：输入 "My Web Server"
   - 主机字段：输入示例 IP "192.168.1.100"
   - 端口字段：保持默认 "22"
   - 用户名字段：输入 "root"
   - 点击 [下一步]

3. **步骤二：认证方式** (2:30 - 3:15)
   - 展示三种认证选项
   - 选择 "SSH Agent（推荐）"
   - 显示检测到的 agent 和密钥
   - 演示切换到 "密码认证"
   - 输入密码（显示为圆点）
   - 切换回 SSH Agent 选项
   - 点击 [下一步]

4. **步骤三：选择分组** (3:15 - 3:45)
   - 显示现有分组（目前只有未分组）
   - 选择 "创建新分组"
   - 输入 "Production"
   - 点击 [完成]

5. **完成添加** (3:45 - 4:00)
   - 对话框关闭
   - 新服务器出现在 Production 分组
   - 状态图标显示 🟢（绿色，表示正常）

**[旁白]**
"点击'添加服务器'，会打开三步向导。第一步填写服务器基本信息：名称、主机地址、端口和用户名。"

"第二步选择认证方式。有三种选择："
- "SSH Agent 是最推荐的，它使用系统已配置的密钥，安全又便捷"
- "SSH 密钥是直接指定私钥文件"
- "密码认证是最简单但相对不安全的方案"

"我们会选择 SSH Agent，稍后教程会详细讲解如何配置密钥。"

"第三步选择分组。分组帮助你组织多台服务器。我们创建一个名为 Production 的分组。"

"点击完成，服务器就添加到列表中了。绿色状态图标表示连接正常。"

**[字幕]**
- 步骤一：基本信息
- 步骤二：认证方式（SSH Agent 推荐）
- 步骤三：分组管理

### 【连接到服务器】(4:00 - 5:30)

**[章节标题卡]**
"建立连接"

**[画面]**
1. **连接方式** (4:00 - 4:30)
   - 方式一：双击服务器项
   - 方式二：点击右侧 [▶] 按钮
   - 高亮两种方式

2. **唤起终端** (4:30 - 4:50)
   - 双击服务器
   - 屏幕边缘显示原生终端窗口打开
   - 终端中自动输入 SSH 命令

3. **连接成功** (4:50 - 5:15)
   - 终端显示连接过程
   - 显示服务器欢迎信息
   - 显示命令提示符 root@my-server:~#
   - 演示执行简单命令：ls, pwd

4. **连接失败场景** (5:15 - 5:30)
   - 快速展示错误情况（用于教学）
   - 状态图标变为 🔴 红色
   - 鼠标悬停显示错误提示

**[旁白]**
"连接服务器非常简单。你可以双击服务器项，或者点击右侧的连接按钮。"

"EasySSH Lite 不会内置终端模拟，而是唤起你系统上安装的原生终端。这样你能获得最佳的终端体验。"

"看到连接成功的提示，说明一切正常。你可以像平常一样使用 SSH 会话。"

"如果连接失败，服务器状态图标会变红。鼠标悬停可以看到错误信息，帮助排查问题。常见原因包括网络不通、认证失败、服务器离线等。"

**[字幕]**
- 双击或点击 ▶ 按钮连接
- 唤起原生终端
- 状态图标指示连接状态

### 【管理多台服务器】(5:30 - 7:00)

**[章节标题卡]**
"使用分组管理"

**[画面]**
1. **批量添加** (5:30 - 5:50)
   - 快速演示添加 2-3 台服务器（快进）
   - 界面现在显示多台服务器
   - 部分在 Production 分组，部分在未分组

2. **创建分组** (5:50 - 6:10)
   - 进入设置 → 分组管理
   - 点击 [+ 新建分组]
   - 输入 "Staging"
   - 点击确定

3. **移动服务器** (6:10 - 6:35)
   - 返回主界面
   - 拖拽演示：
     - 按住服务器项
     - 拖拽到 Staging 分组标题
     - 释放，服务器移动
   - 批量移动演示：
     - Ctrl + 点击选中多台
     - 右键 → 移动到分组
     - 选择 Staging

4. **展开/折叠** (6:35 - 6:50)
   - 点击分组标题 ▼Production
   - 分组折叠，显示 ▶Production
   - 再次点击展开
   - 按 Alt + 0 折叠所有
   - 再按展开所有

5. **搜索过滤** (6:50 - 7:00)
   - 点击搜索框
   - 输入 "web"
   - 只显示名称含 web 的服务器
   - 按 Esc 清除搜索

**[旁白]**
"当你的服务器数量增加时，分组功能非常重要。"

"在设置中可以创建新分组。创建后，你可以通过拖拽将服务器移动到不同分组。按住 Ctrl 可以多选，批量移动。"

"点击分组标题可以展开或折叠，快速切换视图。快捷键 Alt + 0 可以一键展开或折叠所有分组。"

"搜索栏支持实时过滤。输入关键词，只显示匹配的服务器。支持搜索名称、主机地址、用户名。"

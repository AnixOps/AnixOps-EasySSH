# EasySSH Lite 版本规划

> SSH配置保险箱 - 极简、安全、原生

---

## 1. 版本定位

### 核心价值
- **极简主义**：功能列表控制在10项以内
- **原生优先**：不做终端模拟，专注配置管理
- **零信任安全**：默认加密，最小暴露

### 目标用户
- 注重隐私的独立开发者/运维
- 少量服务器（<10台）
- 追求原生终端体验
- 不需要云同步

### 成功指标
- 安装包 < 5MB
- 首次启动 < 2秒
- 内存占用 < 50MB
- 零崩溃率

---

## 2. 功能规格

### 核心功能 (8项)

| # | 功能 | 优先级 | 技术实现 |
|---|------|--------|----------|
| 1 | 服务器CRUD | P0 | SQLite本地 |
| 2 | 服务器分组 | P0 | 单层分组 |
| 3 | Keychain集成 | P0 | keyring crate |
| 4 | 一键连接 | P0 | 原生终端唤起 |
| 5 | 搜索过滤 | P1 | 前端即时搜索 |
| 6 | 导入~/.ssh/config | P1 | 解析OpenSSH格式 |
| 7 | 主密码保护 | P1 | Argon2id + AES-256-GCM |
| 8 | 主题切换 | P2 | 深色/浅色 |

### 禁止添加的功能
- 内置终端模拟
- 分屏
- 云同步
- 监控小组件
- 插件系统

### 配置加密

```
~/.config/easyssh-lite/
├── config.json          # 应用配置（非敏感）
├── connections.enc      # 加密的服务器配置 (Argon2id + AES-256-GCM)
└── groups.json          # 分组配置（明文，无敏感信息）
```

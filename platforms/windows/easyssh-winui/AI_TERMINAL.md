# AI智能终端功能

EasySSH Windows版本现已集成AI智能终端助手，提供强大的AI辅助功能，让您的SSH会话更加高效和安全。

## 功能概述

### 1. AI命令补全 (Command Completion)
- **实时建议**: 在您输入命令时提供智能补全建议
- **上下文感知**: 基于您的命令历史和当前上下文进行预测
- **混合模式**: 结合规则匹配和AI模型生成建议

### 2. 智能错误诊断 (Error Diagnosis)
- **自动分析**: 当命令出错时自动分析错误原因
- **解决方案**: 提供具体的问题修复步骤
- **预防措施**: 建议如何避免类似错误

### 3. 自然语言转命令 (Natural Language)
- **口语化输入**: 用自然语言描述需求，AI转换为Shell命令
- **示例**: 输入"查看内存使用情况" → 输出"free -h"
- **多操作系统支持**: 根据目标系统生成合适的命令

### 4. AI命令解释器 (Command Explainer)
- **工作原理**: 解释命令的各个部分和它们的作用
- **安全提醒**: 标识潜在的危险操作
- **学习资源**: 提供相关文档链接和示例

### 5. 智能日志分析 (Log Analyzer)
- **错误模式识别**: 自动识别日志中的错误和警告模式
- **统计汇总**: 提供错误频率和趋势分析
- **修复建议**: 基于日志内容提供解决方案

### 6. AI安全审计 (Security Audit)
- **危险命令检测**: 识别rm -rf /等危险操作
- **权限检查**: 提醒需要root权限的命令
- **安全替代方案**: 提供更安全的命令替代选项

## 使用方法

### 启用AI助手
1. 点击工具栏中的 `🧐` (AI Assistant) 按钮
2. AI助手面板将显示在右侧

### 自然语言转命令
1. 在AI面板中选择 "Ask" 标签
2. 用自然语言描述您想做的事情
3. 点击 "Execute" 获取生成的命令

### 命令补全
1. 在终端中开始输入命令
2. 按 `Tab` 键触发补全
3. 使用方向键选择建议

### 错误诊断
1. 当命令执行出错时，点击 "Fix" 标签
2. 错误输出会自动填入
3. 点击 "Execute" 获取诊断结果

### 安全审计
1. 在输入敏感命令前，点击 "Audit" 标签
2. 输入要审计的命令
3. 查看风险等级和建议

## AI提供商配置

### 支持的提供商
- **Claude** (Anthropic) - 推荐
- **OpenAI** (GPT-4/GPT-3.5)
- **本地模型** (Ollama/llama.cpp) - 隐私优先

### 配置方法
1. 点击AI面板右上角的 `⚙️` (设置) 按钮
2. 选择AI提供商类型
3. 输入API密钥（如使用云端API）
4. 点击 "Save" 保存配置
5. 点击 "Test Connection" 测试连接

### 隐私模式
- 启用隐私模式后将使用本地模型
- 不会向云端发送任何数据
- 需要本地运行Ollama或llama.cpp

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+A` | 切换AI助手面板 |
| `Tab` | 触发命令补全 |
| `Ctrl+Shift+E` | 解释当前命令 |
| `Ctrl+Shift+D` | 诊断最近错误 |

## 安全提示

1. **自动审计**: 所有执行的命令都会经过安全审计
2. **高危命令确认**: rm -rf /等危险命令会要求额外确认
3. **数据隐私**: 使用本地模型时数据不会离开您的机器
4. **API密钥**: 请妥善保管您的API密钥，不要共享给他人

## 故障排除

### AI助手无法连接
- 检查网络连接
- 验证API密钥是否正确
- 尝试切换AI提供商
- 检查防火墙设置

### 建议不准确
- 提供更多上下文信息
- 使用更具体的自然语言描述
- 尝试不同的AI提供商
- 更新AI模型版本

### 性能问题
- 启用本地模型以提高响应速度
- 减少同时打开的AI会话数量
- 定期清理AI缓存

## 技术实现

### 架构
```
AI Terminal
├── Providers (OpenAI/Claude/Local)
├── Context Tracker (会话历史)
├── Suggestion Engine (建议引擎)
└── UI Components (用户界面)
```

### 缓存机制
- 命令补全结果缓存5分钟
- 安全审计结果缓存5分钟
- 自动清理过期缓存

### 隐私保护
- 敏感数据在发送到云端前会被过滤
- 支持完全离线的本地模型
- 所有历史数据本地存储

# 教程 05：快捷键使用

> 预计时间：5分钟  |  难度：入门

## 目标

完成本教程后，你将学会：
- 掌握所有 EasySSH Lite 快捷键
- 使用键盘快速导航和操作
- 自定义快捷键绑定
- 提高效率的操作技巧

---

## 为什么使用快捷键？

### 鼠标 vs 键盘

| 操作 | 鼠标操作 | 快捷键 | 节省时间 |
|------|----------|--------|----------|
| 添加服务器 | 点击 → 点击 → 点击 | Ctrl+N | 70% |
| 连接服务器 | 移动 → 双击 | Enter | 80% |
| 搜索服务器 | 点击搜索框 → 输入 | Ctrl+F | 50% |
| 展开分组 | 点击分组标题 | Alt+1 | 60% |
| 切换主题 | 设置 → 外观 → 主题 | Ctrl+T | 90% |

**每天使用 EasySSH Lite 30 次的用户：**
- 全鼠标操作：约 5 分钟
- 全快捷键操作：约 1.5 分钟
- **每天节省 3.5 分钟，一年节省 21 小时！**

| 快捷键 | Windows/Linux | macOS | 功能 |
|--------|---------------|-------|------|
| 添加服务器 | `Ctrl + N` | `Cmd + N` | 打开添加服务器对话框 |

| 切换主题 | `Ctrl + T` | `Cmd + T` | 深色/浅色主题切换 |
| 退出应用 | `Alt + F4` | `Cmd + Q` | 退出 EasySSH Lite |

### 导航快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `↑` / `↓` | 上下移动选择 | 在服务器列表中移动 |
| `←` / `→` | 展开/折叠分组 | 在有子项时有效 |
| `Enter` | 连接选中服务器 | 等同于双击 |
| `Space` | 展开/折叠分组 | 选中分组标题时 |
| `Alt + 1~9` | 展开/折叠第N个分组 | 快速切换分组 |
| `Alt + 0` | 展开/折叠所有分组 | 一键展开/收起 |
| `Ctrl + Home` | 跳到列表顶部 | 快速导航 |
| `Ctrl + End` | 跳到列表底部 | 快速导航 |

### 服务器操作快捷键

| 快捷键 | Windows/Linux | macOS | 功能 |
|--------|---------------|-------|------|
| 编辑服务器 | `Ctrl + E` | `Cmd + E` | 编辑选中服务器 |

| 复制连接命令 | `Ctrl + Shift + C` | `Cmd + Shift + C` | 复制 SSH 命令到剪贴板 |

| 移动到分组 | `Ctrl + Shift + M` | `Cmd + Shift + M` | 打开移动分组菜单 |

### 搜索快捷键

| 快捷键 | Windows/Linux | macOS | 功能 |
|--------|---------------|-------|------|
| 聚焦搜索 | `Ctrl + F` | `Cmd + F` | 光标移动到搜索框 |
| 清除搜索 | `Esc` | `Esc` | 清空搜索内容 |
| 下一个结果 | `Enter` / `↓` | `Enter` / `↓` | 选择下一个匹配项 |
| 上一个结果 | `Shift + Enter` / `↑` | `Shift + Enter` / `↑` | 选择上一个匹配项 |

### 分组管理快捷键

| 快捷键 | 功能 |
|--------|------|

| `Ctrl + Shift + ↓` | 将服务器移到下一个分组 |

## 步骤详解

### 步骤 1：基本导航练习

**场景：** 有 3 个分组，每个分组有若干服务器

```
当前界面：
┌─────────────────────────────────────────────┐
│  EasySSH Lite                               │
├─────────────────────────────────────────────┤
│  🔍 搜索服务器...                          │
├─────────────────────────────────────────────┤
│                                             │
│  ▼ Production (3)                          │
│    ├─ 🟢 web-1                             │
│    ├─ 🟢 web-2                             │
│    └─ 🟢 db-1                              │
│                                             │
│  ▶ Staging (2)       ← 已折叠             │
│                                             │
│  ▶ Development (4)   ← 已折叠             │
│                                             │
├─────────────────────────────────────────────┤
│  [+ 添加服务器]              [⚙ 设置]       │
└─────────────────────────────────────────────┘
```

**练习操作：**

1. **使用方向键导航**
   ```
   按 ↓ 键：
   - 第1次：选中 Production 分组标题
   - 第2次：选中 web-1
   - 第3次：选中 web-2
   - 第4次：选中 db-1
   - 第5次：选中 Staging 分组标题
   ```

2. **展开/折叠分组**
   ```
   选中 Staging 分组时：
   - 按 → 键或 Space 键：展开分组
   - 显示内部服务器
   - 再按 ← 键或 Space 键：折叠分组
   ```

3. **快速展开所有分组**
   ```
   按 Alt + 0：
   - 所有分组展开，显示所有服务器
   再按 Alt + 0：
   - 所有分组折叠
   ```

---

### 步骤 2：快速连接服务器

**场景：** 需要快速连接到 web-1

**方法一：纯键盘操作**

```
1. Ctrl + F          → 聚焦搜索框
2. 输入 "web-1"      → 自动过滤显示 web-1
3. Enter             → 连接 web-1

总耗时：约 2 秒
```

**方法二：导航选择**

```
1. ↓↓               → 选中 Production 分组
2. →                → 展开分组
3. ↓                → 选中 web-1
4. Enter            → 连接 web-1

总耗时：约 3 秒
```

**方法三：数字快捷键**

```
1. Alt + 1          → 展开 Production（假设是第1个分组）
2. ↓                → 选中 web-1
3. Enter            → 连接 web-1

总耗时：约 2 秒
```

---

### 步骤 3：高效搜索技巧

**基础搜索：**

```
Ctrl + F
输入: web

结果:
▼ Production (2)
  ├─ 🟢 web-1
  └─ 🟢 web-2

（隐藏不匹配的服务器）
```

**高级搜索语法：**

| 语法 | 示例 | 效果 |
|------|------|------|
| `host:` | `host:192.168` | 按 IP 地址筛选 |
| `user:` | `user:root` | 按用户名筛选 |
| `group:` | `group:prod` | 按分组筛选 |
| `status:` | `status:offline` | 按状态筛选 |

**示例：**

```
搜索栏输入: group:prod user:admin

结果：仅显示 Production 分组中用户名为 admin 的服务器
```

---

### 步骤 4：批量操作

**场景：** 将 Production 中的 3 台服务器移动到 Staging

```
1. 选中 Production 分组
2. 按 → 展开分组
3. 按住 Ctrl（或 Cmd），依次点击：
   - web-1
   - web-2
   - db-1
4. 选中后按 Ctrl + Shift + M
5. 弹出分组选择菜单，选择 Staging
6. 所有选中的服务器移动到 Staging
```

**快捷键总结：**

```
批量选择：Ctrl + 点击（多选）
批量移动：Ctrl + Shift + M
```

### 步骤 5：自定义快捷键

**打开自定义设置：**

1. 设置 → 快捷键

```
┌───────────────────────────────────────────────────────┐
│  设置                                                  │
├─────────────────┬─────────────────────────────────────┤
│  通用           │  快捷键                              │
│  外观           │                                      │
│  安全           │  ┌────────────────────────────────┐  │
│  快捷键      ←  │  │  操作           当前绑定        │  │
│                 │  ├────────────────────────────────┤  │
│                 │  │  添加服务器     Ctrl+N         │  │
│                 │  │  [修改] [清除]  [恢复默认]      │  │
│                 │  ├────────────────────────────────┤  │
│                 │  │  搜索           Ctrl+F         │  │
│                 │  │  [修改] [清除]  [恢复默认]      │  │
│                 │  ├────────────────────────────────┤  │
│                 │  │  ...                           │  │
│                 │  └────────────────────────────────┘  │
│                 │                                      │
│                 │  [重置所有快捷键]                    │
│                 │  [导出快捷键配置]                    │
│                 │  [导入快捷键配置]                    │
│                 │                                      │
└─────────────────┴─────────────────────────────────────┘
```

**修改快捷键：**

1. 点击要修改操作右侧的 **[修改]** 按钮
2. 按下新的快捷键组合（如 `Ctrl + Shift + N`）
3. 如果与现有快捷键冲突，会提示冲突信息
4. 确认后新快捷键生效

**推荐自定义：**

| 操作 | 默认 | 建议自定义 | 理由 |
|------|------|----------|------|
| 连接服务器 | Enter | 保持 | 直观 |
| 复制SSH命令 | Ctrl+Shift+C | Ctrl+Shift+C | 保持 |
| 快速搜索 | Ctrl+F | Ctrl+K | 类似 VS Code |
| 主题切换 | Ctrl+T | Ctrl+Shift+T | 避免冲突 |

## 效率技巧

### 技巧 1：快速切换最近连接

```
快捷键：Ctrl + R（Recent）

弹出最近连接列表：
┌─────────────────────────────────────┐
│  最近连接              [清除历史]   │
├─────────────────────────────────────┤
│  1. web-1 (Production)    2分钟前   │
│  2. db-1 (Production)     15分钟前  │
│  3. test-server (Staging) 1小时前   │
├─────────────────────────────────────┤
│  直接按 1/2/3 快速连接              │
└─────────────────────────────────────┘
```

### 技巧 2：快速编辑模式

```
选中服务器后：
- 按 F2：快速编辑服务器名称
- 按 Ctrl + E：完整编辑对话框
- 按 Delete：删除（带确认）
```

### 技巧 3：搜索后批量操作

```
1. Ctrl + F
2. 输入 "prod" → 筛选出所有生产服务器
3. Ctrl + A → 全选筛选结果
4. Ctrl + Shift + M → 移动到分组
```

### 技巧 4：键盘导航分组

```
Alt + 数字键：快速展开第N个分组

示例：
Alt + 1 → 展开第1个分组（Production）
Alt + 2 → 展开第2个分组（Staging）
Alt + 3 → 展开第3个分组（Development）
```

### 技巧 5：快速复制信息

```
选中服务器后：
- Ctrl + Shift + C：复制完整 SSH 命令
  结果: ssh root@192.168.1.10 -p 22

- Ctrl + Shift + H：复制 user@hostname
  结果: root@192.168.1.10

- Ctrl + Shift + I：复制 IP 地址
  结果: 192.168.1.10
```

## 平台差异说明

### Windows 特定

```
全局搜索：Win + S 可调出系统搜索
Alt + F4：退出应用
Ctrl + Insert：复制
Shift + Insert：粘贴
```

### macOS 特定

```
Cmd + ,：设置（Apple 标准）
Cmd + Q：退出（Apple 标准）
Cmd + C/V：复制/粘贴（与 Linux/Win 一致）
Cmd + [：返回上一页
Cmd + ]：前进下一页
```

### Linux 特定

```
Ctrl + Shift + C/V：终端复制/粘贴
Alt + F4：退出
Super（Win键）：调用系统菜单
```

## 常见问题

### Q1: 快捷键无效

**排查步骤：**

```
1. 检查是否有其他软件占用快捷键
   - 输入法切换（如 Ctrl+Space）
   - 截图软件
   - 其他系统快捷键

2. 在设置中重置快捷键
   设置 → 快捷键 → [重置所有快捷键]

3. 重启 EasySSH Lite
```

### Q2: 如何查看所有快捷键？

**方法：**

```
1. 设置 → 快捷键（完整列表）

2. 快捷键：Ctrl + / 或 Cmd + /
   弹出快捷键速查面板

3. 帮助菜单 → 快捷键参考
```

### Q3: 能否完全禁用快捷键？

```
设置 → 快捷键 → [禁用所有快捷键]
或自定义模式：仅保留基础导航快捷键
```

### Q4: 如何导出/导入快捷键配置？

```
导出：设置 → 快捷键 → [导出快捷键配置]
      保存为 .json 文件

导入：设置 → 快捷键 → [导入快捷键配置]
      选择之前保存的 .json 文件

用途：
- 备份自定义配置
- 团队统一快捷键设置
- 多设备同步
```

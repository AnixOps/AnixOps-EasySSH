# EasySSH v0.3.0 Beta Release Notes | EasySSH v0.3.0 Beta 发布说明

> **Version | 版本**: v0.3.0-beta.1
> **Codename | 代号**: Horizon
> **Release Date | 发布日期**: 2026-04-03

---

## Release Overview | 版本概述

EasySSH v0.3.0-beta.1 "Horizon" marks our first public beta release, introducing a revolutionary three-tier product strategy designed to meet the diverse needs of individual developers, power users, and enterprise teams.

EasySSH v0.3.0-beta.1 "Horizon" 是我们的首个公开 Beta 版本，推出了革命性的三版本产品战略，旨在满足个人开发者、高级用户和企业团队的多样化需求。

This release features native UI implementations across all platforms - Windows (egui), Linux (GTK4), and macOS (SwiftUI) - ensuring optimal performance and native user experience on each operating system.

本版本在各平台均采用原生 UI 实现 - Windows (egui)、Linux (GTK4) 和 macOS (SwiftUI) - 确保每个操作系统上的最佳性能和原生用户体验。

---

## Editions Available | 可用版本

### Lite Edition | Lite 版本

**Positioning | 定位**: SSH Config Vault | SSH 配置保险箱

The Lite edition focuses on secure SSH configuration management with native terminal integration. Perfect for privacy-conscious developers who prefer working with their system's native terminal.

Lite 版本专注于安全的 SSH 配置管理和原生终端集成。非常适合注重隐私、喜欢使用系统原生终端的开发者。

**Core Features | 核心功能**:
- Native terminal launch | 原生终端唤起
- Secure credential storage | 安全凭据存储
- Server grouping (single level) | 服务器分组（单层）
- Quick search and filter | 快速搜索过滤

---

### Standard Edition | Standard 版本

**Positioning | 定位**: Full SSH Client | 全功能 SSH 客户端

The Standard edition builds upon Lite with an embedded terminal, split-screen layout, and real-time monitoring - ideal for power users managing multiple servers.

Standard 版本在 Lite 基础上增加了嵌入式终端、分屏布局和实时监控 - 非常适合管理多台服务器的高级用户。

**Core Features | 核心功能**:
- All Lite features | Lite 所有功能
- Embedded terminal with WebGL acceleration | 嵌入式终端（WebGL 加速）
- Split-screen layout | 分屏布局
- SFTP file management | SFTP 文件管理
- Real-time monitoring widgets | 实时监控小组件
- Auto-reconnect | 自动重连

---

### Pro Edition | Pro 版本

**Positioning | 定位**: Team Collaboration Platform | 团队协作平台

The Pro edition extends Standard with team management, audit logging, and SSO integration - designed for IT teams and enterprises requiring centralized SSH management.

Pro 版本在 Standard 基础上增加了团队管理、审计日志和 SSO 集成 - 专为需要集中式 SSH 管理的 IT 团队和企业设计。

**Core Features | 核心功能**:
- All Standard features | Standard 所有功能
- Team management | 团队管理
- Audit logging | 审计日志
- SSO integration (SAML/OIDC) | SSO 集成（SAML/OIDC）
- RBAC permissions | RBAC 权限控制
- Shared snippets | 共享代码片段

> **Note | 注意**: Pro edition features are currently in preview mode with limitations: max 3 teams, max 10 members per team, 30-day audit log retention.
>
> Pro 版本功能目前处于预览模式，有限制：最多 3 个团队，每团队最多 10 名成员，审计日志保留 30 天。

## Key Features | 关键功能

### Native UI | 原生用户界面

| Platform | Technology | Features | 特点 |
|----------|------------|----------|------|
| Windows | egui | Pure Rust, lightweight & efficient | 纯 Rust 实现，轻量高效 |
| Linux | GTK4 | Native GNOME, Wayland support | GNOME 原生体验，Wayland 支持 |
| macOS | SwiftUI | Apple native, deep integration | 苹果原生框架，深度系统集成 |

### Security | 安全特性

| Feature | Description |
|---------|-------------|
| Argon2id | Memory-hard password hashing for key derivation |
| AES-256-GCM | Authenticated encryption for data at rest |
| Keychain Integration | Secure credential storage using OS keychain |
| E2EE | End-to-end encryption for Pro edition sync |

| 功能 | 说明 |
|------|------|
| Argon2id | 内存硬化密码哈希用于密钥派生 |
| AES-256-GCM | 认证加密保护静态数据 |
| Keychain 集成 | 使用操作系统密钥链安全存储凭据 |
| E2EE | Pro 版本同步的端到端加密 |

### SSH Key Management | SSH 密钥管理

- Support for RSA, ED25519, ECDSA key types | 支持 RSA、ED25519、ECDSA 密钥类型
- SSH Agent forwarding | SSH Agent 转发
- Key passphrase protection | 密钥密码保护
- Import from `~/.ssh/config` (experimental) | 从 `~/.ssh/config` 导入（实验性）

### Connection Management | 连接管理

- Server grouping with drag-and-drop | 服务器分组支持拖拽
- Connection history with timestamps | 连接历史带时间戳
- Quick connect via search | 搜索快速连接
- Batch operations (Standard+) | 批量操作（Standard 及以上）

### Pro Edition Specific | Pro 版本特定问题

| Issue | Severity | Planned Fix |

- **ssh2 crate**: RSA 密钥解析 CVE 等待上游 v0.9.4 修复

## Installation Instructions | 安装说明

### Windows

**System Requirements | 系统要求**:
- Windows 10 version 1809 or later | Windows 10 版本 1809 或更高
- 64-bit (x64) | 64 位 (x64)

**Download | 下载**:
```
https://easyssh.io/downloads/easyssh-lite-0.3.0-beta.1-x64.msi
https://easyssh.io/downloads/easyssh-standard-0.3.0-beta.1-x64.msi
https://easyssh.io/downloads/easyssh-pro-0.3.0-beta.1-x64.msi
```

**Installation Steps | 安装步骤**:
1. Download the MSI installer for your edition | 下载对应版本的 MSI 安装包
2. Double-click to run the installer | 双击运行安装程序
3. Follow the setup wizard | 按照安装向导操作
4. Launch EasySSH from Start Menu | 从开始菜单启动 EasySSH

### Linux

**System Requirements | 系统要求**:
- Ubuntu 22.04+ / Fedora 37+ / Arch Linux (rolling)
- GTK4 4.6+ | GTK4 4.6+

**Download | 下载**:
```
# DEB package (Ubuntu/Debian)
https://easyssh.io/downloads/easyssh-lite-0.3.0-beta.1-amd64.deb
https://easyssh.io/downloads/easyssh-standard-0.3.0-beta.1-amd64.deb
https://easyssh.io/downloads/easyssh-pro-0.3.0-beta.1-amd64.deb

# RPM package (Fedora/RHEL)
https://easyssh.io/downloads/easyssh-lite-0.3.0-beta.1-x86_64.rpm
https://easyssh.io/downloads/easyssh-standard-0.3.0-beta.1-x86_64.rpm
https://easyssh.io/downloads/easyssh-pro-0.3.0-beta.1-x86_64.rpm

# AppImage (universal)
https://easyssh.io/downloads/easyssh-lite-0.3.0-beta.1-x86_64.AppImage
https://easyssh.io/downloads/easyssh-standard-0.3.0-beta.1-x86_64.AppImage
https://easyssh.io/downloads/easyssh-pro-0.3.0-beta.1-x86_64.AppImage
```

**Installation Steps (DEB) | 安装步骤 (DEB)**:
```bash
sudo dpkg -i easyssh-lite-0.3.0-beta.1-amd64.deb
sudo apt-get install -f  # Install dependencies
```

**Installation Steps (AppImage) | 安装步骤 (AppImage)**:
```bash
chmod +x easyssh-lite-0.3.0-beta.1-x86_64.AppImage
./easyssh-lite-0.3.0-beta.1-x86_64.AppImage
```

### macOS

**System Requirements | 系统要求**:
- macOS 12.0 (Monterey) or later | macOS 12.0 (Monterey) 或更高
- Apple Silicon (M1/M2) and Intel supported | 支持 Apple Silicon (M1/M2) 和 Intel

**Download | 下载**:
```

https://easyssh.io/downloads/easyssh-lite-0.3.0-beta.1-arm64.dmg

https://easyssh.io/downloads/easyssh-pro-0.3.0-beta.1-x64.dmg
```

**Installation Steps | 安装步骤**:
1. Download the DMG file | 下载 DMG 文件
2. Open the DMG and drag EasySSH to Applications | 打开 DMG 并将 EasySSH 拖到应用程序文件夹
3. Right-click and select "Open" on first launch (Gatekeeper) | 首次启动时右键点击并选择"打开"（Gatekeeper）

## Upgrade Notes | 升级说明

### First Beta Release | 首个 Beta 版本

This is the first public beta release of EasySSH. No upgrade is required if you are a new user.

这是 EasySSH 的首个公开 Beta 版本。如果您是新用户，无需升级。

### Migrating from Development Builds | 从开发版本迁移

If you have been testing development builds, please note:

如果您一直在测试开发版本，请注意：

1. **Data Backup | 数据备份**: Back up your configuration before upgrading | 升级前请备份您的配置
   - Windows: `%APPDATA%\EasySSH\`
   - Linux: `~/.config/easyssh/`
   - macOS: `~/Library/Application Support/EasySSH/`

2. **Database Migration | 数据库迁移**: v0.3.0-beta.1 uses a new database schema. Migration will be automatic on first launch | v0.3.0-beta.1 使用新的数据库架构。首次启动时将自动迁移

3. **Configuration Reset | 配置重置**: If you encounter issues, try resetting to defaults | 如果遇到问题，请尝试重置为默认值

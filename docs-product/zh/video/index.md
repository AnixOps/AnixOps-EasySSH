## 录屏技术指南

### 录制设置

**macOS:**
```bash
# 使用 ScreenFlow 或 OBS
# 建议设置:
# - 分辨率: 2560x1440 (录制) → 1920x1080 (输出)
# - 帧率: 60fps
# - 格式: ProRes 或高质量 H.264
```

**Windows:**
```powershell
# 使用 OBS Studio
# 建议设置:
# - 基础分辨率: 1920x1080
# - 输出分辨率: 1920x1080
# - 录像格式: mkv (防止损坏)
# - 编码器: NVENC (NVIDIA) 或 AMF (AMD)
```

**Linux:**
```bash
# 使用 OBS 或 SimpleScreenRecorder
# 命令行方案:
ffmpeg -f x11grab -r 60 -s 1920x1080 -i :0.0 \
  -c:v libx264 -preset fast -crf 18 output.mkv
```

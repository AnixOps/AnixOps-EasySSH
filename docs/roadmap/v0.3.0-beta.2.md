## 5. Terminal Integration Architecture

```
┌─────────────────────────────────────────────────────────┐
│  Frontend (WebView2/wry)                                │
│  ┌─────────────────────────────────────────────────────┐│
│  │ xterm.js + WebGL addon                              ││
│  │ - xterm-addon-webgl (GPU rendering)                 ││
│  │ - xterm-addon-fit (auto resize)                     ││
│  │ - xterm-addon-search (find)                         ││
│  └─────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────┘
                         │ WebView IPC
                         ▼
┌─────────────────────────────────────────────────────────┐
│  Rust Backend (easyssh-core)                            │
│  ┌─────────────────────────────────────────────────────┐│
│  │ TerminalCoordinator                                 ││
│  │ - TabManager (multitab.rs)                          ││
│  │ - PtyTerminal (embedded.rs)                         ││
│  │ - SshSessionManager (ssh.rs)                        ││
│  └─────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────┘
                         │ SSH2 Protocol
                         ▼
┌─────────────────────────────────────────────────────────┐
│  Remote Server (sshd + shell)                           │
└─────────────────────────────────────────────────────────┘
```

### Key Components

| Component | Location | Purpose |
|-----------|----------|---------|
| TerminalCoordinator | easyssh-core | Unified coordinator |
| PtyTerminal | embedded.rs | PTY abstraction |
| TabManager | multitab.rs | Tab state management |
| WebGlTerminal | easyssh-winui | xterm.js integration |
| StreamingProcessor | streaming.rs | Big data handling |

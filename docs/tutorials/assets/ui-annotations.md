## 主界面元素

### 整体布局

```
┌────────────────────────────────────────────────────────────────┐
│ [A]                      [B]                    [C]    [D]      │
├────────────────────────────────────────────────────────────────┤
│ [E]                                                            │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  [F] Production (3)                                          │
│    ├─ [G] [H] [I] [J]                              [K]        │
│    ├─ [G] [H] [I] [J]                              [K]        │
│    └─ [G] [H] [I] [J]                              [K]        │
│                                                                │
│  [F] Staging (1)                                              │
│    └─ [G] [H] [I] [J]                              [K]        │
│                                                                │
├────────────────────────────────────────────────────────────────┤
│  [L]                              [M]                          │
└────────────────────────────────────────────────────────────────┘
```

### 元素详解

| 标识 | 名称 | 类型 | 功能 | 交互 |
|------|------|------|------|------|
| A | 应用标题 | Label | 显示 "EasySSH Lite" | 无 |
| B | 窗口控制 | Button | 最小化/最大化/关闭 | 点击 |
| C | 设置菜单 | Button | 打开设置面板 | 点击 [≡] |
| D | 主题切换 | Button | 切换深色/浅色主题 | 点击 [☀/🌙] |
| E | 搜索框 | Input | 搜索过滤服务器 | 输入文字 |
| F | 分组标题 | Header | 显示分组名称和数量 | 点击展开/折叠 |
| G | 状态图标 | Icon | 显示服务器状态 | 无 |
| H | 服务器名称 | Label | 显示服务器名称 | 无 |
| I | 连接信息 | Label | 显示 user@host | 无 |
| J | 端口信息 | Label | 显示端口（非22时） | 无 |
| K | 连接按钮 | Button | 一键连接 | 点击 [▶] |
| L | 添加按钮 | Button | 打开添加对话框 | 点击 [+ 添加服务器] |
| M | 设置按钮 | Button | 打开设置面板 | 点击 [⚙ 设置] |

### 状态图标说明

| 图标 | 状态 | 颜色 | 含义 |
|------|------|------|------|
| 🟢 | online | #4CAF50 | 连接正常 |
| 🟡 | warning | #FFC107 | 警告/维护中 |
| 🔴 | offline | #F44336 | 离线/连接失败 |
| ⚫ | unknown | #9E9E9E | 状态未知 |
| 🔵 | connecting | #2196F3 | 正在连接 |

## 添加服务器对话框

### 步骤一：基本信息

```
┌──────────────────────────────────────────────────────────────┐
│  添加服务器                                        [关闭] X   │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  ○ 步骤 1    ● 步骤 2    ○ 步骤 3                           │
│  [基本信息]  [认证方式]  [选择分组]                          │
│                                                              │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  名称*       [A]                                             │
│              [Web Server 1___________________]              │
│              给服务器起个便于识别的名字                      │
│                                                              │
│  主机*       [B]                                             │
│              [192.168.1.10___________________]              │
│              IP地址或域名                                    │
│                                                              │
│  端口        [C]                                             │
│              [22_____________________________]              │
│              默认SSH端口为22                                 │
│                                                              │
│  用户名*     [D]                                             │
│              [admin__________________________]                │
│              登录用户名                                        │
│                                                              │
├──────────────────────────────────────────────────────────────┤
│  [E] 取消                        [F] 下一步 →               │
└──────────────────────────────────────────────────────────────┘
```

| 标识 | 名称 | 类型 | 必填 | 验证规则 |
|------|------|------|------|----------|
| A | 名称 | TextInput | 是 | 1-50字符，唯一 |
| B | 主机 | TextInput | 是 | 有效IP或域名 |
| C | 端口 | NumberInput | 否 | 1-65535，默认22 |
| D | 用户名 | TextInput | 是 | 1-32字符 |
| E | 取消按钮 | Button | - | 关闭对话框 |
| F | 下一步 | Button | - | 进入步骤二 |

| H | 下一步 | Button | 进入步骤三 |

## 标注样式规范

### 颜色定义

```css
:root {
  /* 主要颜色 */
  --annotation-primary: #FF4444;    /* 重要元素 */
  --annotation-secondary: #4488FF;  /* 次要元素 */
  --annotation-tertiary: #44AA44;   /* 提示信息 */
  --annotation-neutral: #888888;    /* 中性元素 */

  /* 背景 */
  --annotation-bg: rgba(255, 255, 255, 0.9);
  --annotation-border: #333333;

  /* 字体 */
  --font-number: 'Arial Black', sans-serif;
  --font-text: 'Arial', sans-serif;
}
```

### 标注尺寸

| 元素 | 尺寸 | 字重 |
|------|------|------|
| 标注数字 | 24px | Bold |
| 标题文字 | 18px | Bold |
| 说明文字 | 14px | Regular |
| 辅助文字 | 12px | Regular |

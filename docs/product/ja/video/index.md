## スクリーン録画技術ガイド

### 録画設定

**macOS:**
```bash
# ScreenFlow または OBS を使用
# 推奨設定:
# - 解像度: 2560x1440 (録画) → 1920x1080 (出力)
# - フレームレート: 60fps
# - フォーマット: ProRes または高品質 H.264
```

**Windows:**
```powershell
# OBS Studio を使用
# 推奨設定:
# - ベース解像度: 1920x1080
# - 出力解像度: 1920x1080
# - 録画フォーマット: mkv (破損防止)
# - エンコーダー: NVENC (NVIDIA) または AMF (AMD)
```

**Linux:**
```bash
# OBS または SimpleScreenRecorder を使用
# コマンドラインオプション:
ffmpeg -f x11grab -r 60 -s 1920x1080 -i :0.0 \
  -c:v libx264 -preset fast -crf 18 output.mkv
```

### ポストプロダクション

**編集の推奨事項:**
1. 不要な一時停止を削除
2. 遅い操作を高速化（説明を追加）
3. 重要なインターフェイス要素を拡大
4. 注釈とハイライトを追加

**字幕仕様:**
- 日本語: 源ノ角ゴシック Regular
- 英語: Inter Regular
- フォントサイズ: 48pt (1080p)
- 位置: 下部中央

**サウンドエフェクト:**
- クリック: 軽い「カチッ」音
- 成功: 軽快な通知音
- エラー: 低い通知音（耳障りでない）
